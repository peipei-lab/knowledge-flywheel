# DECISION RULES

## Source Selection

- User-curated sources are high-signal until feedback proves otherwise.
- A source is valuable when it changes a question, decision, or draft direction.
- Preserve why a source was selected; that reason is preference data.

## Draft Selection

- A draft is worth developing when it has a clear mechanism, lived relevance, and audience value.
- A draft should not be published only because the source is interesting.
- Repeated revision patterns should become explicit writing principles.

## Memory Promotion

- Raw feedback is the source of truth.
- Promoted principles are provisional working models.
- Conflicting feedback should remain visible until resolved by future reviews.
